Diese Single Page App ermöglicht es dem Anwender, süße und herzhafte Rezepte
rund um die Weihnachtszeit in sechs Kategorien (Kuchen, Plätzchen, Dessert, Vorspeise,
Hauptspeise und Fingerfood) mit Rezeptbild, Zubereitungszeit, Kurzbeschreibung, Zutatenliste, Merkmalen (vegan, vegetarisch, laktosefrei oder glutenfrei) und Titel in eine Google-Firebase-Datenbank hochzuladen, und sie sich anschließend anzeigen zu lassen.
Dabei wurde diese Single Page App nach dem Mobile First Prinzip entwickelt. Die Voraussetzung zur Nutzung dieser Single Page App ist, dass das Engerät internetfähig und einen Browser auf dem Endgerät installiert ist.

Die Anzeige kann über folgende Möglichkeiten erfolgen:
Es ist möglich, sich alle vorhandenen Rezepte auf der Startseite unsortiert anzeigen zu lassen.
Außerdem gibt es ein Menü, über das sich der Anwender nur die Rezepte anzeigen lassen kann, die der ausgewählten Kategorie entspricht.
Zudem ist eine Suchfunktion implementiert, in der es möglich ist, nach Schlagwörtern zu suchen und sich nur die Rezepte anzeigen zu lassen, die das Schlagwort enthalten.
Als letzte Möglichkeit gibt es außerdem einen Filter, der Rezepte nach Zubereitungszeit oder bestimmten Merkmalen (vegan, vegetarisch, laktosefrei, glutenfrei), anzeigt.

Nachdem eine der oben definierten Methoden benutzt wurde, wird vorerst ausschließlich der Titel des Rezepts und das zugehörige Bild angezeigt. Durch das Anklicken des Bildes, wird der Anwender zur Detailansicht des Rezepts geführt, die als eine weitere Seite der Single Page App implementiert wurde. Diese enthält zusätzlich zu Titel und Bild noch die Zubereitungszeit, die Kurzbeschreibung und die Zutaten.

Damit dieses Projekt in Atom eingebunden werden kann, muss das Repository dieses Projektes in einem entsprechenden Dateipfad mit dem PC verbunden werden.

Anleitung für PCs mit Windows-Betriebssystem:
Hierfür öffnet man erste einmal den Windows-Explorer und navigiert in den Dateipfad, in dem später das Projekt abgelegt werden soll. Anschließend ersetzt man den Dateipfad oben in der leiste mit "cmp". Daraufhin erscheint die Benutzeroberfläche von Windows. In diese Benutzeroberfläche gibt man folgende Kommandozeile ein: git clone git@github.com:matze121413/web-programmierung_gruppenprojekt.git. Anschließend bestätigt man die Kommandozeile mit ENTER.
Nun kann das Projekt über Atom eingebunden werden: File -> Open Folder -> Projekt auswählen.
